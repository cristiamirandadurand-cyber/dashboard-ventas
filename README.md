# Dashboard de Ventas de Equipos Móviles
Este proyecto consiste en un dashboard interactivo desarrollado en Power BI para el análisis de ventas.
**Características del proyecto:**
* Análisis de indicadores clave de rendimiento (KPIs).
* Visualización de tendencias de ventas.
* Limpieza y modelado de datos en Power BI.

**Cómo verlo:**
Descarga el archivo ".pbix" del repositorio y ábrelo con Power BI Desktop.
